## 支持emoji输入法适用的盘白

> 💡 **提示**：快速技巧或建议

> ⚠️ **注意**：可能遇到的问题

> ❗️ **重要**：必须遵守的规则

> 🚫 **严禁**：在生产环境直接运行！

> 🔥 **高危操作**：执行前请**务必备份**！

> ☠️ **危险**：此操作会导致**数据永久丢失**！

## 支持输入文本符号的适用旁白

有点废格子。

**提示**

```
\---------------

■ 提示：基础信息

\---------------
```

**注意**

```
\-----------

◆ 注意：操作细节

\-----------
```

**重要**

```
\-----------

● 重要：核心步骤  

\-----------
```

**警告**

```
\--------------

▲ 警告：安全事项

\--------------
```

**严禁**

```
\----------------

‼ 严禁：禁止行为

\----------------
```

## 适用于不方便输入符号的旁白

### 方案一：方括号和汉字。

不太适用于多段文字内容

[提示]具体内容...[提示完]

增强版为数字标记，比原始版本略强一点。

【1提示】内容...【1结束】

### 方案二：级别对应

**提示/标记**

```
--[提示/标记]--

...

--[结束]--
```

**重要**

```
==【重要】==

...

==【结束】==
```

**警告/注意**

```
==警告/注意==

...

==结束==
```

**危险**

```
==!危险!==

...

==!结束!==
```

## 使用html标签进行旁白标注

缺点：代码复杂；优点：多样性。

* 波浪线：<span style="text-decoration: wavy underline;color: red;">波浪线</span>，继续普通文本。
* 下划线：<u>下划线文字</u>
* 文字下的着重点：这是<span style="text-emphasis: dot; text-emphasis-position: under;">文字下方着重点</span>。


```
# 波浪线，字体红色
<span style="text-decoration: wavy underline; color: red;">波浪线</span>


# 文字下方着重点
<span style="text-emphasis: dot; text-emphasis-position: under;">文字下方着重点</span>
```


```
/* 语法：线型 + 样式 + 颜色 + 厚度 */
text-decoration: underline wavy red 2px;
            ↓         ↓       ↓    ↓
          "装饰"  "在哪里" "什么样" "什么色" "多粗"
```
